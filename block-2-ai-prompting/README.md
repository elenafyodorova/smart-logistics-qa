# Блок 2: Проектирование тестов — AI Prompting & Review

Чек-лист для тестирования формы создания заявки на перевозку («УЛ Лайт»), сгенерированный с помощью LLM, с авторским ревью результата и анализом граничных значений.

## Структура файлов

- `prompt-and-raw-response.md` — промпт и необработанный (raw) ответ LLM
- `review-and-boundary-values.md` — ревью ответа ИИ (что упущено/избыточно) и граничные значения для тестирования

## Контекст

Форма заявки на перевозку: город отправления, город назначения, вес груза (кг, до 20 000), объём груза (м³, до 82), дата погрузки (датапикер, без прошедших дат), кнопка «Рассчитать стоимость» (POST-запрос к API).
