# QA Engineer Test Assignment — Smart Logistics

Репозиторий с выполнением тестового задания на позицию QA Engineer (Web/Mobile, 1C & AI-Assisted).

## Структура

```text
├── README.md
├── block-1-etrn/
├── block-2-ai-prompting/
├── block-3-sql/
├── block-4-api/
└── block-5-1C/
```

Каждый блок содержит отдельный README с описанием решения и используемых материалов.

## Выполненные задания

- Блок 1 — Тестирование требований (ЭТрН)
- Блок 2 — Проектирование тестов (AI Prompting & Review)
- Блок 3 — Анализ данных (SQL)
- Блок 4 — Тестирование API
- Блок 5 — Базовые вопросы по 1С

## Использование AI

В заданиях, где это предусмотрено условиями, использовались LLM-инструменты. Для таких задач сохранены:
- исходный промпт;
- ответ модели;
- собственный анализ и замечания.
