# Блок 3: Анализ данных (SQL)

Решения SQL-задач на диалекте PostgreSQL: вторая по величине зарплата, поиск дублирующихся email, клиенты без заказов.

## Структура файлов

- `solutions.sql` — запросы с краткими пояснениями к каждому решению
- `prompt-engineering.md` — дополнительная практика: промпты для генерации SQL-запросов с помощью LLM на самостоятельно собранной тестовой базе логистической тематики, проверка результатов в DBeaver
