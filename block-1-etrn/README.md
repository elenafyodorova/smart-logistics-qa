# Блок 1: Тестирование требований — Бизнес-процесс ЭТрН

Анализ бизнес-процесса электронной транспортной накладной (ЭТрН), чек-лист/тест-кейсы по основному процессу, негативные сценарии и баг-репорты.

## Структура файлов

- `business-process-analysis.md` — анализ бизнес-процесса на основе BPMN-диаграммы
- `etrn-checklist.md` — чек-лист / тест-кейсы: создание, валидация полей, смена статусов, отображение реквизитов, обработка ошибок
- `negative-scenarios.md` — негативные проверки с ожидаемым поведением системы
- `bug-reports.md` — баг-репорты

## Источники

- BPMN-диаграмма процесса ЭТрН
- XSD-схема электронных транспортных накладных (nalog.gov.ru)
